หลังจากที่ได้ทำ self-practice สัปดาห์แรกไปก็ได้ความรู้เรื่อง javascript มากขึ้นโดยส่วนใหญ่ที่เขียนในโค้ดจะเกี่ยวกับการเขียน function
โดยเรื่องที่ได้เรียนรู้มาเพิ่มมากขึ้นจะเป็นเรื่อง การใช้ method, การกำหนดตัวแปร, expression, typeof, spread operator, object
หัวข้อที่เขียนก็จะเน้นการใช้ expression, medthod, การเขียน function กำหนดตัวแปรซะส่วนใหญ่ เพราะคิดว่าเป็นสิ่งที่จำเป็นในการเขียน
javascript มากๆ หลังจากได้เขียนมาทั้งหมดสองสัปดาห์ท้ายก็รู้สึกได้ว่าตัวเองมีพื้นฐานมากขึ้นในการเขียน javascript ถึงแม้อาจจะ
ไม่ได้เขียนในช่วงสามสัปดาห์แรกแต่ก็จะพยายามฝึกด้วยตัวเองต่อไปเรื่อยๆ จะต้องเขียน javascript คล่องมากขึ้นแน่นอน
